Computer Vision Week8: Texture and shape

Texture 材质

从图片处理的角度来看，

材质是具有空间平稳性（spatial stationarity）的区域（region）（该区域的任何地方都具有相同的统计性质）

从图形学（computer graphics）角度来看，

材质是可以应用在3D模型上的2D表面。

材质种类

  1. Periodic Texture （有重复规则pattern的，has a subregion that repeats in a
    regular pattern）

  1. Stocastic （aperiodic ）texture（有随机步骤生成的）

材质模型

做材质模型的意义是为了创造更多可用于计算机图形、视频游戏等的纹理 ，图像修复的材质。

还可以做材质转移(texture transfer)，用于艺术效果，网络购物。（感觉就是能生成一些更仿真的样子？）

  1. Parametric models：用一组可调参数表示材质

  2. Non parametric (stitching) models: 由图片拼接成的材质

无参数材质生成（synthesis）

Image stitching approach

当图片中出现missing pixel的时候，可以通过观察missing pixel 所在patch周围的情况，找到和当前patch相似的patches，然后填补missing pixel。

这是一种Non-parametric texture approach

  1. 从原始图像中随机采样一个小的（例如，3 x 3 像素）补丁
  2. 向外旋转（Sprial outward，我理解是螺旋样一层一层往外找？），通过在原始纹理中找到相似的Neighbourhoods来填充缺失的像素（Neighborhood size是一个free parameter，它指定纹理的随机程度)

Image quilting 图片绗缝

2001年发表的高效patch-based 材质生成方法。使用现有的纹理补丁来合成更多的纹理；

主要问题是将它们连接在一起的时候却没有可见的人工接缝痕迹

也叫做 “Corrupt Professor’s Algorithm"，尽可能多地“抄袭”源图像，掩盖抄袭证据。

  1. 选择patch和overlap 大小。
  2. 用随机patch进行初始化
  3. 针对每一个subsequent patch，在原始纹理中找到与该区域最相似的patch，仅考虑重叠区域中的pixels， 确保overlap error达到最小化后，将此patch无缝粘贴（seamlessly paste）过去

Image Impainting 图像修复

和填补missing region的思路一致，从另外一张图里找到相似的patch，确保错误最小化之后粘贴到原图片。

参数材质生成 Parametric texture synthesis

用多个参数表示材质

为了合成纹理，强制noisy image 连接所需的参数（通常通过梯度下降）

Fourier texture synthesis 傅里叶材质生成

Matching Fourier magnitude 来生成材质

可以生成简单材质结果，但是无法生成困难材质

复杂的材质可以被定义成由一个简单材质分布（distribution over simple features）演化而来的。

然而简单的feature distribution 往往不够生成完整的材质，因此还额外需要表达出feature的co-occurrence。

总结

  • 非参数材质合成基于复制texture patches

  • 适合周期性材质，periodic textures

  • 缺点：没有texture parameter 模型

  • 参数材质合成根据一组参数表示材质

  • 大多数方法在随机纹理上效果更好

  • 缺点：即便使用非常复杂的模型（神经网络）也可能导致结果不完整

Texture Transfer

两个图像（内容、样式）都通过在 ImageNet 上训练的 VGG 网络运行

content 表示为来自神经网络层的response

style表示为神经网络层的特征图之间的correlation

使用gradient descent 找到匹配内容和风格的图像。

Loss是由content reconstruction和style reconstruction的总和

content和style的相对权重是一个可以控制成像的free parameter

通常情况下，content match存在于高层 layer里，style match存在与全部layer里。

Shape 形状

2D 形状模型通常基于

  1. 形状的边界轮廓（线段、角度）（Bounding contour）
  2. 形状的内部结构（分支）(interal structure)

Shape Skeleton

Topological skeleton = thinnest possible version of a shape, 图像最细的形状

由与形状边界等距的线组成

骨架点是可以放入形状内的最大圆的中心点

如果我们将形状看作是用可变半径的圆形画笔画成的，那么骨架则是画笔的路径。

Grassfire transform algorithm 是一种缩小或者细化形状的算法（shrinking or thinning a shape）每一个在shape内的pixel，计算与之最近的boundary的距离。计算完成后与boundary距离最远的则是skeleton。

Skeleton + 每个像素距离boundary的距离可以看成shape的紧凑的、可逆（compact，invertible）表达形式。

Invertible的意思是，如果想用skeleton 绘制图像，只需要在每个skeleton pixel上放一个圆盘即可。

shape skeleton 很容易转换为graph

graph 表示可用于shape matching 或者pose recognition

shape skeleton也可以将2D平面图变为3D，只需要将填充的圆盘变为球即可。

Shape 缺点

  • 形状必须要从背景分割开来 segmented from background

  • Boundary的改变也会改变skeleton

Contour representations 轮廓表示

轮廓表示是一个可以fit 自身boundary的parametric model。

用“shrink wraps” （收缩包装？类似抽真空袋？）来包裹住物体从而取得物体的轮廓。

Active contour algorithm

  1. 首先在物体boundary外初始化轮廓
  2. 接下来的在轮廓上的每一个pixel都要在每一步向任意方向移动一格像素
  4. 重复以上步骤直到loss 不再改变。

EelasticityE_{elasticity}Eelasticity​ 基于轮廓长度，penalize 较长的轮廓

EstiffnessE_{stiffness}Estiffness​ 基于轮廓曲率，直线轮廓线段最低 （straight contour segments）

EedgeE_{edge}Eedge​ 基于轮廓位置的图像梯度，图像梯度最高的位置最低

Active contours 经常用于segementation和 tracking，在医学影像分析中很常见。

Active contours 缺点

  • 需要（人为介入的）初始化
  • 并不能保证完美契合图形
  • trade off between elasticity & edge
  • 很难检测杂乱（clutter）的形状

总结

  • 人脸模型是计算机视觉中形状表示的主要应用之一
  • 目前最先进的算法基于 3D 人脸模型
  • 应用：
    • 面部识别
    • 计算机图形（电影 CGI、游戏）
    • Zoom 滤镜
  • 尽管Shape不需要category 级别的物品识别（category-level object recognition），但是在pose、lighting上仍有一定的要求。
  • 2D模型经常用于骨架结构或边界轮廓 （skeleton structure or bounding contours）
  • 3D模型经常常用于脸部识别等识别问题。
