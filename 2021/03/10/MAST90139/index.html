2. Handling Qualitative predictor or Categorical factor

Qualitative is category

Quantitative is numerical

当 Qualitative predictor (categorical predictor) 用于model的时候，我们必须将它变为 factor。

A qualitative predictor has to be coded into a set of numerical variable called contrast

在model中qualitative predictors，每一个level都要被coded 到 model里。

也就是说有k level的factor，最后需要有k-1个dummy variable to display if this level is present or not.

注意：当model有interaction term的时候，也要注意dummy variable。
